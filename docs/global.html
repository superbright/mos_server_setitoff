(constant) APPSTATE :number

Enum for game state.
Type:
  • number
Properties:
Name Type Description
SETUP number Guest page is active, gameplay page is inactive. Can be set from any state.
READY number Guest page is active, gameplay page is active. Can only be set from SETUP.
INGAME number Guest page is inactive, gameplay page is active. Can only be set from READY.
END number Broadcasts game is in END state then automatically set to SETUP.
RESET number Broadcasts game is in RESET state then automatically set to SETUP.
